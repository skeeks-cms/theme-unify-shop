Installation
------------

The preferred way to install this extension is through [composer](http://getcomposer.org/download/).

Either run

```
php composer.phar require --prefer-dist skeeks/theme-unify-shop "^0.0.1"
```

or add

```
"skeeks/theme-unify-shop": "^0.0.1"
```
